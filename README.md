## 打包&运行

需求环境:
- nodejs7+

```bash
$ npm install

$ npm run dev # 运行开发服务器
#or
$ npm run build # 打包js文件
#or
$ npm run package # 打包当前平台的pc端文件
```

**开发服务器访问`http://127.0.0.1:8080/`即可访问客户端**  
**客户端需要配合服务端才能正常运行, 可以添加环境变量`NODE_ENV=production`访问官方服务器**
